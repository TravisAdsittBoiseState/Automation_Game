# Automation Game

Automation Game is a simple console application game created to practice and demonstrate OOP principles in C++.

## Compiling/Running

This was developed in Visual Studio(VS) 2019 Community, thus should be opened as a solution in VS and compiled/ran using that IDE.

Possible make file compilation support in the future.

## Usage

When running the program you have the option to send commands to a rover or have the rover execute the commands it already has. Follow the prompts to play, a rovers valid commands are: 
'w' - Move up 
'a' - Move left
's' - Move down
'd' - Move right
'c' - Collect resources (if any are nearby)
